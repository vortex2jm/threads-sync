# Threads sync work :gear:

## Usage
 - Compiling programs
 ```bash
 make
 ```
 - Running monkeys problem first part
 ```bash
make run-monkey
 ```
 - Running with gorillas
 ```bash
 make run-gorillas
 ```
 - Running monitors problem
 ```bash
 make run-monitors
 ```
 - Removing temporary files
 ```bash
 make clean
 ```
